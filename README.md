## Installation

To install the project in your development environment ❤️

```bash
  git clone "https://github.com/otavioaraujo08/nendoroid-colection.git"
```

Installing the dependencies 💀

```bash
  npm install or npm i
```

Starting server 🚀

```bash
  npm start
```
